1. CDATA, PCDATA
   - xml 파서 : xml문서를 읽어들여 해석하는 프로그램
   - PCDATA : xml 문서에서 xml 파서에 의해 처리될 부분
   			  따로 지정하지 않으면 xml 문서 전체
   - CDATA : xml 파서가 처리할 필요가 없는 부분
   			 mapper파일안에 sql은 xml이 아니다 -> <![CDATA[ ~~~ ]]> (xml 파일이 아니다 표시해주는 것)
   - mapper에서 CDATA 처리가 꼭 필요한 경우
   		sql에 < 문자가 포함된 경우 (태그의 시작으로 인식하기때문에)
   			<![CDATA[ ~~~ ]]> (태그의 시작이 아니야 라고 인식시켜주거나)
   			&lt;로 처리할 수도 있다
